5 मिनट में OpenClaw जैसा व्यक्तिगत AI एजेंट प्राप्त करें। सैंडबॉक्स सुरक्षित, macOS और Windows के लिए अति-छोटा 30MB इंस्टॉलर (1 मिनट में इंस्टॉल)। WhatsApp, Telegram, Slack, Discord, Gmail, DingTalk, WeChat Work, QQ, Feishu और अन्य मैसेजिंग ऐप्स से कनेक्ट होता है। बिल्ट-इन स्किल मार्केट, नॉलेज बेस, मेमोरी, MCP, शेड्यूल्ड टास्क। Go में विकसित: तेज़ और कम संसाधन उपयोग।

## प्रीव्यू

### AI चैट असिस्टेंट

अपने AI असिस्टेंट से कोई भी प्रश्न पूछें, और यह प्रासंगिक उत्तर देने के लिए आपके नॉलेज बेस को बुद्धिमान तरीके से खोजेगा।

![](../../images/1.png)

### PPT त्वरित निर्माण

स्मार्ट असिस्टेंट को एक वाक्य का कमांड भेजें और स्वचालित रूप से PowerPoint प्रस्तुति बनाएं।

![](../../images/2.png)

### स्किल मैनेजर

कमांड का उपयोग करें ताकि रोबोट आपको अपने कंप्यूटर पर इंस्टॉल की गई सुविधाओं को खोजने या नए एक्सटेंशन प्लगइन इंस्टॉल करने में मदद करे।

![](../../images/3.png)

### नॉलेज बेस | डॉक्यूमेंट वेक्टराइज़ेशन स्टोरेज

अपने डॉक्यूमेंट अपलोड करें (जैसे TXT, PDF, Word, Excel, CSV, HTML, Markdown)। सिस्टम स्वचालित रूप से पार्स, स्प्लिट और वेक्टर एम्बेडिंग में कन्वर्ट करेगा, आपके निजी नॉलेज बेस में स्टोर करेगा ताकि AI मॉडल सटीक रिट्रीवल और उपयोग कर सके।

![](../../images/4.png)

### तत्काल Q&A के लिए टेक्स्ट सिलेक्शन

अपनी स्क्रीन पर कोई भी टेक्स्ट चुनें, यह स्वचालित रूप से कॉपी होगा और फ्लोटिंग क्विक-ask बॉक्स में भर जाएगा। एक क्लिक में, तत्काल उत्तर के लिए AI असिस्टेंट को भेजें।

![](../../images/5.png)

![](../../images/6.png)

### स्मार्ट स्नैप विंडो

एक स्मार्ट असिस्टेंट जो अन्य ऐप्लिकेशन विंडो के बगल में स्नैप हो सकता है। प्रश्न पूछने के लिए इसमें विभिन्न कॉन्फ़िगर किए गए AI असिस्टेंट के बीच तेज़ी से स्विच करें। रोबोट आपके कनेक्टेड नॉलेज बेस के आधार पर उत्तर देता है और आपकी बातचीत में एक क्लिक में रिप्लाई भेजने का समर्थन करता है।

![](../../images/7.png)

### एक प्रश्न, एकाधिक उत्तर: आसानी से तुलना करें

अपने प्रश्न को दोहराने की आवश्यकता नहीं है। एक साथ कई "AI विशेषज्ञों" से परामर्श करें और एक ही इंटरफेस में उनके उत्तरों को साथ-साथ देखें। तुलना आसान है और आपको सर्वश्रेष्ठ निष्कर्ष पर पहुंचने में मदद करता है।

![](../../images/8.png)

### वन-क्लिक लॉन्चर बॉल

ChatClaw मुख्य ऐप्लिकेशन विंडो को तुरंत जगाने या खोलने के लिए अपने डेस्कटॉप पर फ्लोटिंग बॉल पर क्लिक करें।

## सर्वर मोड परिनियोजन

ChatClaw सर्वर मोड में चला सकता है (डेस्कटॉप GUI की आवश्यकता नहीं), ब्राउज़र के माध्यम से सुलभ।

### बाइनरी सीधे चलाएं

अपने प्लेटफॉर्म के लिए बाइनरी [GitHub Releases](https://github.com/chatwiki/chatclaw/releases) से डाउनलोड करें:

|| प्लेटफॉर्म | फ़ाइल |
||----------|------|
|| Linux x86_64 | `ChatClaw-server-linux-amd64` |
|| Linux ARM64 | `ChatClaw-server-linux-arm64` |

```bash
chmod +x ChatClaw-server-linux-amd64
./ChatClaw-server-linux-amd64
```

अपने ब्राउज़र में http://localhost:8080 खोलें।

सर्वर डिफ़ॉल्ट पर `0.0.0.0:8080` पर सुनता है। आप पर्यावरण चर के माध्यम से होस्ट और पोर्ट कस्टमाइज़ कर सकते हैं:

```bash
WAILS_SERVER_HOST=127.0.0.1 WAILS_SERVER_PORT=3000 ./ChatClaw-server-linux-amd64
```

### Docker

```bash
docker run -d \
  --name chatclaw-server \
  -p 8080:8080 \
  -v chatclaw-data:/root/.config/chatclaw \
  registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
```

अपने ब्राउज़र में http://localhost:8080 खोलें।

### Docker Compose

`docker-compose.yml` फ़ाइल बनाएं:

```yaml
services:
  chatclaw:
    image: registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
    container_name: chatclaw-server
    volumes:
      - chatclaw-data:/root/.config/chatclaw
    ports:
      - "8080:8080"
    restart: unless-stopped

volumes:
  chatclaw-data:
```

फिर चलाएं:

```bash
docker compose up -d
```

अपने ब्राउज़र में http://localhost:8080 खोलें। रोकने के लिए: `docker compose down`। डेटा `chatclaw-data` वॉल्यूम में रहता है।
